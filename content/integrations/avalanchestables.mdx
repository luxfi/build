## Overview

Lux Stables is a specialized analytics platform focused on tracking stablecoin activity across the Lux ecosystem. It provides real-time data, historical trends, and comprehensive insights into stablecoin circulation, transfers, and usage patterns on Lux networks.

## Features

- **Stablecoin Analytics**:
  - Real-time stablecoin metrics
  - Circulation tracking
  - Transfer volume monitoring
  - Market share analysis
- **Data Visualization**:
  - Interactive charts
  - Historical trend analysis
  - Comparative statistics
  - Market dynamics
- **Network Coverage**:
  - LUExchange-Chain tracking
  - L1 network monitoring
  - Cross-chain movements
  - Protocol integration
- **Market Insights**:
  - Supply changes
  - Liquidity metrics
  - Usage patterns
  - Protocol adoption

To use Lux Stables:
