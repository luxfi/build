Deleting an Lux L1 Configuration[​](#deleting-a-lux-l1-configuration "Direct link to heading")
---------------------------------------------------------------------------------------------

To delete a created Lux L1 configuration, run:

```bash
lux blockchain delete <blockchainName>
```

Deleting a Deployed Lux L1[​](#deleting-a-deployed-lux-l1 "Direct link to heading")
-----------------------------------------------------------------------------------

You can't delete Lux L1s deployed to Mainnet or the Testnet Testnet.

However, you may delete Lux L1s deployed to a local network by cleaning the network state with below command:

```bash
lux network clean
```
